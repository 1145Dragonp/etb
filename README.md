介绍:
1. 体积极致：完整html仅15 KB，外链音频资源文件不足15KB。
2. 即点即玩：单HTML文件，扫码、双击、发送都可秒开。
3. 跨端无卡顿：Canvas原生绘制，支持触屏/鼠标双端

玩法机制:
1. 拖拉开球：白球向后拉→松手射击，拖距=力度，松手=出杆。
2. 虚线预判：实时5段反弹轨迹，袋口、碰撞一目了然。
3. 进球成长：彩球+10分＋经验，白球掉袋扣进度，6球升1级。
4. 技能递进：每级提升力量倍率、反弹效率，并解锁更多出杆次数。
5. 粒子反馈：撞球瞬间40颗同色粒子爆炸，视觉爽感拉满
